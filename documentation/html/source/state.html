Documentation State¶

class automata.State(name, final=False)¶

Bases: object

Base module node for an Automata graph. Connects to other nodes, which ultimately evaluate the acceptance of a string inputed by the user

Parameters

  • __name__ (str) – The name of the State. This is used for identification purposes

  • __transitions ([[State, str]]) – The connections inbetween nodes. Forms a connection between one node and the next. The transition to the next State object is taken if the first character of the string is contained in the __transitions[x][1] string

  • __final (bool) – Checks wether the State is a final State, meaning that if no further characters are in the string, if the result will be true or false

evaluate(string)¶

Recursive function which recursively goes through the transition graph of each node, determining if when all characters in the input string are parsed, if the graph evaluates the given input as True (if parsing ends in a __final State) or otherwise False. The function prints the graph path taken in the console.

Parameters

string (str) – The String which is to be recursively parsed throughout the graph
